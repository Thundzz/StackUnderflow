In order to start using StackUnderflow :

1/ Install the gems specified in the Gemfile with:
   $ bundle install
2/ Create or update the database with:
   $ rake db:migrate
3/ Run the server with:
   $ rails s

You can now access to the website following this URL:
http://localhost:3000
